## This repository contains the R and Python code that was used to analyze the manuscript "Single-cell eQTL mapping reveals immune cell type-specific genetic modulation in children and adolescents with type 1 diabetes"


## This repository includes scripts for multiple genomics analysis tasks:
- **scRNA-seq preprocessing**
- **Cell proportion estimation / analysis**
- **Pseudobulk expression generation**
- **HLA analysis**
- **eQTL preprocessing, analysis, and visualization**
- **Clinical variance analysis**
- **Genetic risk score (GRS) optimization**

## The `demo/` folder contains example input files for eQTL analysis.

## Requirements
- Python (Scanpy v1.9.3, statsmodels v0.13.5, DoubletDetection v4.2, Harmony v0.0.6)
- R (Matrix eQTL v2.3)
